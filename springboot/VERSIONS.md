# Configurações do Ambiente de Desenvolvimento

Este documento descreve as versões recomendadas para o desenvolvimento do projeto Java/Spring Boot.

## Requisitos de Software

| Componente | Versão | Link de Download / Referência |
| :--- | :--- | :--- |
| **Java JDK** | 17 em diante | [Download the Latest Java LTS Free](https://www.oracle.com/java/technologies/downloads/) |
| **Maven** | 4.0.0 em diante | [Apache Maven Downloads](https://maven.apache.org/download.cgi) |
| **Spring Boot** | 3.0.0 em diante | [Spring Initializr](https://start.spring.io/) |
| **MySQL** | 8.0 em diante | [MySQL Installer for Windows](https://dev.mysql.com/downloads/installer/) |
| **IDE** | IntelliJ IDEA | *Opcional* |

---
*Nota: Certifique-se de configurar as variáveis de ambiente (JAVA_HOME, MAVEN_HOME) corretamente após a instalação.*
